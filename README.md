# AJU - Anti Judol Detector

AJU (Anti Judol Detector) adalah sebuah ekstensi browser yang dirancang untuk membantu pengguna mendeteksi dan memblokir konten yang terkait dengan judi online (judol).

## Fitur
- **Deteksi Otomatis**: Memindai konten halaman web secara real-time.
- **Whitelist**: Mengecualikan situs terpercaya seperti mesin pencari dari pemindaian.
- **Antarmuka Sederhana**: Pengaturan dan status deteksi yang mudah diakses melalui popup.

## Instalasi
1. Clone repositori ini.
2. Buka browser berbasis Chromium (Chrome, Edge, dll).
3. Buka halaman `chrome://extensions/`.
4. Aktifkan **Developer mode** di pojok kanan atas.
5. Klik **Load unpacked** dan pilih folder proyek ini.

## Teknologi
- JavaScript
- HTML/CSS (Manifest V3)
